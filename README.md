# To-Do List App

A simple to-do list application built with HTML, CSS, and JavaScript. This app allows users to add, remove, and check off tasks directly in their browser.

## Features

- Add new tasks
- Remove tasks
- Mark tasks as completed

## Technologies Used

- HTML
- CSS
- JavaScript
